# Projet Scolaire – Polytech Lille ISIA

Ce dépôt présente un ensemble de projets académiques réalisés au sein de Polytech Lille, spécialité ISIA (Informatique, Statistique et Intelligence Artificielle).

Chaque dossier correspond à un projet individuel ou collectif, illustrant la diversité des compétences acquises en programmation, analyse statistique et applications de l’intelligence artificielle durant la formation. 

## Structure du dépôt

- Un dossier par projet, clairement nommé selon la thématique ou l’algorithme abordé.
- Pour chaque projet : code source, rapport technique, éventuels supports complémentaires.

## À propos

Les travaux regroupés ici témoignent de l’application pratique des enseignements reçus et de l’engagement dans l’apprentissage continu des disciplines informatiques et statistiques, avec un intérêt marqué pour l’intelligence artificielle.
